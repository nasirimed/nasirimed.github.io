Privacy Policy for YouTube Comment Search Extension

Last updated: 24 May 2025

Overview

The YouTube Comment Search Extension is designed with privacy as a fundamental principle. Our extension operates entirely locally within your browser and does not collect, store, or transmit any user data.

1. Data Collection and Usage

We want to be explicitly clear about our data practices. Our extension:

  • Does NOT collect any personally identifiable information
  • Does NOT collect health information
  • Does NOT collect financial or payment information
  • Does NOT collect authentication information
  • Does NOT collect personal communications
  • Does NOT collect location data
  • Does NOT collect web history
  • Does NOT track user activity
  • Does NOT store website content

2. How the Extension Works

Our extension:

  • Only reads publicly available YouTube comments on the current page
  • Performs all search operations locally in your browser
  • Does not store or cache any comment data
  • Does not send any data to external servers

3. Required Permissions

The extension requires minimal permissions to function:

  • youtube.com access: To read comment sections on YouTube pages
  • activeTab: To interact with the current YouTube video page
  • tabs: To detect when you're viewing YouTube videos

4. Third-Party Services

We do not:

  • Use any third-party services
  • Include any analytics tools
  • Share any data with third parties
  • Sell or transfer any user data

5. Data Storage

The extension:

  • Does not maintain any databases
  • Does not store any user data locally
  • Does not use browser storage features
  • Does not cache search results between sessions

6. Children's Privacy

Our extension does not collect any information from users of any age, including children under 13.

7. Changes to Privacy Policy

If we make any changes to this privacy policy, we will:

  • Post updates here
  • Update the "Last updated" date
  • Notify users through the Chrome Web Store listing

8. Your Rights

Since we don't collect any personal data, there is no user data to access, modify, or delete. The extension operates purely as a local search tool.
